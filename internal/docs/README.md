# yyork docs

Fumadocs site for yyork product specs, architecture notes, and guides.

From the repo root, docs are opt-in (not started by `pnpm dev` / d3k):

```bash
pnpm dev:docs
```

After the portless ready banner appears, open `https://docs.yyork.localhost`.

`pnpm docs:dev` is an alias of `pnpm dev:docs`.

For direct Vite access (no portless), use `pnpm docs:dev:direct`.
