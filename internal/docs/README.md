# yyork docs

Fumadocs site for yyork product specs, architecture notes, and guides.

From the repo root:

```bash
pnpm dev
```

Docs start alongside the yyork app. After the portless ready banner appears, open `https://docs.yyork.localhost`.

To run docs alone:

```bash
pnpm docs:dev
```

For direct Vite access (no portless), use `pnpm docs:dev:direct`.
